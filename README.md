## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a US Visa dataset to uncover meaningful insights and patterns behind visa approvals and denials.

The goal is to transform raw data into **actionable insights** using data cleaning, visualization, and statistical techniques.

---

## 🎯 Objectives

- Understand the structure and distribution of the dataset  
- Identify key factors influencing visa approval/denial  
- Perform univariate, bivariate, and multivariate analysis  
- Visualize trends using graphs and charts  
- Derive meaningful business insights  

---

## 📂 Dataset Information

The dataset includes various attributes such as:

- Case Status (Approved / Denied)  
- Employer Information  
- Job Category  
- Salary/Wage  
- Work Location  
- Experience & Qualification  

> 📌 *Note: Dataset used is for learning and analysis purposes.*

---

## 🛠️ Tech Stack

- **Programming Language:** Python  
- **Libraries:**  
  - Pandas  
  - NumPy  
  - Matplotlib  
  - Seaborn  
- **Environment:** Jupyter Notebook

## 🔍 Project Workflow

1. **Data Collection**
2. **Data Cleaning & Preprocessing**
3. **Handling Missing Values**
4. **Exploratory Data Analysis (EDA)**
5. **Visualization**
6. **Insights & Conclusion**

---

## 📊 Key Insights

✔️ Certain job categories have higher approval rates  
✔️ Salary plays a significant role in visa approval  
✔️ Some locations show higher denial patterns  
✔️ Employer type impacts approval chances
